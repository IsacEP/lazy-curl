# Lazy Curl

Lazy Curl is a TUI tool and a CLI to enhance and improve your API / REST handeling.
Lazy Curl is an alternative to use normal `Curl` or the graphical `Postman`

## How to run

Start out by downloading this repository:

```
git clone git@github.com:IsacEP/lazy-curl.git
```

After cloning the repository, stand in the repository root and run:

```
go run ./cmd/lazy-curl/
```

You will need Go to run this application.
